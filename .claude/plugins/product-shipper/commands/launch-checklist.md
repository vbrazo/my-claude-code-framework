## Format


```
Launch: <feature/product name>
Date: <target date>
Checklist:
  Engineering: <N>/<total> complete
```


## Rules

- Every checklist item must have a clear owner.
- Do not launch with any engineering items incomplete.
- Infrastructure items can be marked as N/A if not applicable.
